# First Choice Private School Website

## 📖 Project Overview
This project is a simple and responsive website created for First Choice Private School.  
The website provides important information about the school, academic programs, and contact details.

## 🎯 Purpose of the Project
The purpose of this website is to:
- Provide information about the school
- Promote academic excellence
- Improve communication between school and parents
- Practice web development skills using HTML and CSS

## 🏫 About First Choice Private School
First Choice Private School is dedicated to academic excellence, discipline, and character development.  
We aim to provide a safe and supportive learning environment for all students.

## 🌟 Website Features
- Home page with welcome message
- About Us section
- Academic Programs section
- School Projects/Activities section
- Contact Information page
- Clean and user-friendly design

## 🛠 Technologies Used
- HTML
- CSS

## 🚀 How to Use
1. Download the project files.
2. Open the file named `index.html`.
3. View it in any web browser (Chrome, Edge, Firefox).
